# Photos and logo – simple method

**Open your project folder in Finder (or File Explorer).**

Go to: **Residence Amizade** → **public** → **images**

You’re in the right place. The site uses whatever is in that folder.

---

**What to do**

1. **Gallery** – Open the **gallery** folder. There are already files like `exterior-1.jpg`, `garden-1.jpg`, `room-1.jpg`, etc. **Replace those files** with your own photos and keep the **same file names**.

2. **Room photos** – There are 4 folders: `room-standard-single`, `room-standard-double`, `room-superior-double`, `room-family-triple`. Each has 2 images. **Replace those** with your room photos (same names: `room-standard-single-1.jpg`, `room-standard-single-2.jpg`, etc.).

3. **Logo** – Put **logo.png** or **logo.svg** inside the **images** folder (next to the gallery folder).

4. **Hero** (big image at top of home) – Put **hero-image.jpg** in the **images** folder.

No new folders, no renaming. Replace the existing files with yours (same names), or add logo and hero if you have them.
