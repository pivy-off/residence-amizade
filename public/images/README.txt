WHERE TO PUT YOUR PHOTOS AND LOGO
=================================

Open this folder in Finder (Mac) or File Explorer (Windows).
It's inside your project: Residence Amizade > public > images

REPLACE, DON'T CREATE:
- The gallery folder already has placeholder images. Replace them with your photos using THE SAME NAMES (exterior-1.jpg, garden-1.jpg, room-1.jpg, etc.).
- The room-standard-single, room-standard-double, room-superior-double, room-family-triple folders already have 2 images each. Replace those with your room photos (same names).

ADD THESE IF YOU HAVE THEM:
- Logo: put logo.png, logo.svg, or logo.jpg here (in this images folder). The site will use it in the header automatically.
- Hero (big top image): put hero-image.jpg here.
- Video for hero (optional): put hero-video.mp4 here.

That's it. Same folder you're in now. Replace the existing image files with yours, same names.
